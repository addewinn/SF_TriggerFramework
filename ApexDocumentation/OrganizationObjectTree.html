OrganizationObjectTree

public with sharing class OrganizationObjectTree
This class provides a method to generate a custom apex object that groups this organizations SObjects into the following categories: Standard Objects: A wrapper class containing all standard Salesforce SObjects in this org. Custom Objects: A wrapper class containing all custom objects created by this org. Namespaced Objects: A wrapper class containing all SObjects that have a namespace association. Custom Settings: A wrapper class containing all custom settings created by this org. Namespaced Settings: A wrapper class containing all custom settings deployed under a namespace. Use this class with mapping tools to define TradingSchema Settings.

Kevin Winn
2022
